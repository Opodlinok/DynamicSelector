# DynamicSelector
[English version Readme](README.md)

DynamicSelector — инструмент для Unity, предназначенный для создания
параметрических выборщиков и фильтров на основе внешних данных.

Плагин позволяет настраивать логику фильтрации и отображения параметров
через Editor, без жёсткой привязки к коду и без необходимости каждый раз
реализовывать однотипные UI-решения вручную.

---

## Основные возможности

- Создание параметрических выборщиков (фильтры, переключатели, диапазоны)
- Настройка логики через Unity Editor
- Привязка UI к данным из внешних источников
- Модульная архитектура и возможность расширения
- Подходит для мобильных, десктопных и Web-проектов

---

## Для кого этот инструмент

- **Unity-разработчики**, которым нужно быстро реализовать фильтрацию и выбор параметров
- **Дизайнеры**, работающие с UI и конфигурациями без постоянного участия программиста
- **Проекты с динамическими данными**, где структура параметров может меняться со временем

---

## Кратко о том, как это работает

1. Настраивается структура данных (XML фид)
2. В Editor создаётся конфигурация выборщика
3. Параметры сопоставляются с полями данных
4. UI автоматически реагирует на изменения и фильтрацию

---

## Требования

- Unity 2021.3 LTS или новее  
- Поддерживается UI на базе Unity UI / TextMeshPro  
- Проект должен использовать .NET Framework 4.x Equivalent

## Быстрый старт

Импортируйте пакет DynamicSelector в Unity-проект:

**Через Package Manager UI**
1. Откройте **Window → Package Manager**
2. Нажмите **"+" → Add package from git URL...**
3. Введите: ```https://github.com/opodlinok/DynamicSelector.git?path=Packages/src```

**Или вручную**
- Скачайте `.unitypackage` из раздела **Releases** и импортируйте его в проект

**В проекте**
1. Добавьте компонент 'FeedManager' на сцену и заполните его поля (конфигурация)
2. Добавьте компонент 'FilterManager' на сцену, прокиньте его в 'FeedManager' и по необходимости создайте фильтры
3. Создайте префаб оффера с компонентом 'OfferView' и подключите его в 'FeedManager'
4. Сопоставьте поля данных с параметрами UI в 'OfferView'
5. Запустите сцену и настройте поведение

## Обновления

Плагин поддерживает встроенную систему обновлений
(через Toolbar → Window → DynamicSelector → Мастер обновлений).
Также возможна ручная установка обновлений из раздела Releases.
